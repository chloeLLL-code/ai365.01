視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技術，讓網站背景以比前景更慢的速度移動。當訪客向下捲動時，會產生 3D 的視覺效果，增加空間深度並帶來更沉浸的瀏覽體驗。視差源自光學錯覺， 因為人眼會將近處物體視為比遠處物體更大， 因此遠處物體看起來像是移動得更慢。這種錯覺長期被應用在不同媒介的視差效果中， 營造寫實感。它最早用於傳統動畫， 可追溯至迪士尼的《白雪公主與七個小矮人》，以及《超級瑪利歐》等電子遊戲。 隨著 CSS 與 HTML 的發展，視差效果也逐漸演進成我們今天熟悉的網頁設計手法。

早期的瀏覽器視差滾動實作之一，是由網頁開發者 Glutnix 在 2007 年的部落格文章中提出， 內含使用 JavaScript 與 CSS 2 的範例與示範，並支援當時的 Internet Explorer 6 等瀏覽器。 然而直到 2011 年 HTML5 與 CSS 3 的推出，這項技術才在網頁設計師之間變得流行。 支持者認為這是擁抱網頁流動性的一種簡單方式。擁護者將視差背景作為更好地吸引用戶、 提升網站整體體驗的工具。不過，普渡大學於 2013 年發表的研究指出： 「雖然視差滾動提升了使用者體驗的某些面向，但並不一定改善整體使用者體驗」。


為什麼要使用視差圖像？

    設計師應嘗試視差圖像，主要有兩個原因：

  • 有助於說故事

    視差圖像非常適合用來敘事。好的設計在於溝通——當我們設計網站時， 我們希望與訪客溝通，並說出關於產品、服務或品牌的精彩故事。 視差滾動能讓訪客沉浸在獨特的線上體驗中，甚至可以打造單頁式網站， 讓訪客不必切換多個頁面就能閱讀完整故事。

  • 有助於提升使用者參與度

    視差圖像可用於提升使用者參與度。降低網站的跳出率是網頁設計中需要留意的目標之一。 跳出率是指訪客只看一頁就按下「返回」或關閉分頁的比例。 跳出率過高會影響搜尋排名，因此任何能降低跳出率的措施都值得採取。 精心設計的視差頁面能提供動態體驗，往往讓使用者停留更久。


如果你想在網站加入視差滾動效果，請考慮以下建議。

  • 有選擇地使用

    視差滾動在特定元素上使用時更具效果，而不是在所有頁面、所有時間都使用。 可以用在頁首與標題，或只在首頁使用。 你希望視差滾動能強化設計，而不是分散使用者注意力。

  • 圖片壓縮

    視差滾動會使用多種媒體檔案與 CSS 形狀來營造深度感。 你不希望因為過大的圖片檔案載入過慢，而影響網站效能與使用者體驗。

  • 玩轉色彩

    視差滾動不只是前景與背景的圖像，也包含色彩運用。 你可以在背景使用較低飽和度的色調，前景則用更鮮明的色彩， 以創造更強烈的深度感。
